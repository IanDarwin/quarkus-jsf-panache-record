Welcome to the demo app

This is the home page of a trivial demo app showing off the capability of Quarkus to serve as an HTML (JSF) web application host in addition to its better known use as a REST web service engine.

Technologies used include:

  • Quarkus
  • JSF
  • Prime Faces UI add-ons
  • Hibernate
  • Panache
  • CDI
